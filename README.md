# Endless Runner Prototype

A fast-paced endless side-scrolling runner game developed with Unity and C# as part of the Unity Learn pathway.

The player must avoid incoming obstacles by timing jumps correctly while surviving as long as possible in an endless environment.

## Features
- Endless runner gameplay
- Jump and obstacle mechanics
- Repeating background system
- Sound effects and music
- Particle effects
